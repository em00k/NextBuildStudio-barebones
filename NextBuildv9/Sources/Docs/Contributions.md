  - Bearded Themes 
    - Colour themes by BeardedBear
    - https://github.com/BeardedBear/bearded-theme

  - JS AY routines

    - https://github.com/remy/zx-tools

  - Inno Setup 
    - Build tools for deployment 
    - https://jrsoftware.org/isinfo.php

  - Maziac's NEX file viewer  
    - Allows viewing of NEX data structure

    - https://github.com/maziac/z80-instruction-set

  - Maziac's HEX Hover Helper 
    - HEX value hover details 
    - https://github.com/maziac/hex-hover-converter

    - AY sound effect engine by Shiru
